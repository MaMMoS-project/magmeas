magmeas documentation
=====================
This is the documentation of the meagmeas package for import, calculation
and handling of magnetometry data acquired with Quantum Designs VSM
measurement option.

Installation
------------
To use magmeas, simply install it using pip:

.. code-block:: console

   $ pip install magmeas
